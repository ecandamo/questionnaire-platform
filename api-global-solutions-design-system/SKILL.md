## Quick reference

- Brand colors: Navy `#273B6E` (primary), Green `#78BC43` (accent), Ink gray ramp (neutrals)
- Fonts: **Mulish** (display/UI, substitutes Sailec), **Questrial** (docs, substitutes Century Gothic), **JetBrains Mono** (data)
- Icon system: Lucide, 2px stroke, round caps — no emoji, no PNG icons
- Key motif: the "+" sign from the API logo, used large on hero surfaces and as the clear-zone unit
- Token file: `colors_and_type.css` — import it to get every variable
- UI kit: `ui_kits/ops-hub/` — Sidebar, Topbar, KPI, StatusBadge, table patterns

## Primary audience

API customers are **airline operations directors, crew schedulers, dispatchers, and finance controllers** — not consumers. Copy is operational and imperative. Never use emoji or marketing fluff.
